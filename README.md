## 介绍
Biubiubiu是一款免费的BotPro插件，并且长期维护更新
## 功能
插件目前拥有以下功能：
1. 群关键词精确回复
2. 群关键词模糊回复
3. 私聊关键词精确回复
4. 私聊关键词模糊回复
5. 通过发消息教机器人学习
## 插件地址
[https://github.com/BotPro-Plugins/Biubiubiu.git](https://github.com/BotPro-Plugins/Biubiubiu.git)
## 安装方法
下载源码后，到BotPro后台上传插件
上传后打开BotPro插件目录 `app/Plugins`
确保本插件目录名为 `Biubiubiu`
